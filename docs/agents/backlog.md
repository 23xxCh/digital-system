# Backlog Configuration

## Backend

GitHub Issues

## Repository

https://github.com/23xxCh/digital-system

## Workflow

- Create issues: `gh issue create`
- List issues: `gh issue list`
- View issue: `gh issue view <number>`
- Close issue: `gh issue close <number>`
- Add labels: `gh issue edit <number> --add-label <label>`

## Skills that read this file

- `to-issues` — creates issues from specs
- `triage` — processes incoming issues
- `to-prd` — converts context to PRD
- `qa` — reports QA findings
